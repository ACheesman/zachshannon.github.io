Cellular Automaton

May 20, 2016 by Zachary Shannon


Currently WIP.

Appearing some time in the 1970s, John Horton Conway's 'Game of Life' is a so called 'cellular automaton'. Take a base state; by applying some simple rules, that base state 'evolves'. Conway's Game of Life has an even more interesting property though: using Conway's Game of Life we can build a Universal Turing Machine. I am very interested in using Conway's Game of Life to generate 'artistic' images. To this end, I have started work on some Python code to do just that.

This code is not quite ready to produce anything too pretty, and it is a bit slow right now. It does work, though!
